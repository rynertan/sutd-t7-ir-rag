# Charge control microcomputer arrangement for vehicle.

## Abstract
A charging control system for a vehicle 11 includes a charge control microcomputer 6 for determining the most suitable reference voltage for the charging system voltage regulator 3 based on various input data from the charging system and an engine control microcomputer 8 . The device also detects faults in the charging system by processing the charging system data inputs and displays the detection result on a display unit 7 .
