# Magnetic position sensor.

## Abstract
In a magnetic detector 6 for receiving a reference posi tion signal in movement from an encoder track 41, 41A and a reference track 42, 42a, 42A arranged on a member 3, 3A included in a magnetic position sensor, magnetic pickups re spectively confronting to the encoder track and the reference track are provided and both tracks jointly yield a position refer ence signal.
