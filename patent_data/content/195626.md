# Data communications system.

## Abstract
A communication system utulizing the electrical supply wiring of mains circuit 2 has an interface unit 3 for an intruder sensor 1. Unit 3 has a transmitter 4 for sending FSK modulated messages and a capacitor 8 to store ade quate energy to power transmitter 4 only for the time duration of a message. The system also has a interface unit 22 for a temperature sensor 21.
