# Multilayer analysis element.

## Abstract
A dry type multilayer analysis element comprises at least one porous medium layer 50 comprising a membrane filter, to which an antigen or antibody is immobilized, and at least one reagent layer 20 through which a substance s which did not participate in an antigen antibody reaction can permeate. The multilayer analysis element is effective for assaying components present in body fluids, blood, urine, etc., in a simple manner.
