# ICE CREAM MACHINE WITH REMOVABLE VESSEL

## Abstract
The invention relates to an ice cream machine, in particu lar for family use, of the type having a vessel wherein the ice cream is made, which is removable from the machine body. To allow for a good thermal exchange, the machine has a vessel container which is divided into two or more parts, each one comprising cooling coils, said parts being resiliently kept in close contact with the vessel walls in a loosenable manner.
