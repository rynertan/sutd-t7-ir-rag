# Method of character generation from compressed fonts.

## Abstract
Method of character generation for displaying or print ing image data stored sequentially in a compressed font memory. The proper character is addressed during decom pression by combining a calculated approximate pointer 25 obtained by multiplying the code of the character by an approximate pel dimension number, with a limited range correction factor 22, 26 stored with the font to obtain an exact start of character pointer utilized to address the image data for the character. Also stored with the correction factor is data 23 indicating the vertical positioning of the character within the character box 10 .
