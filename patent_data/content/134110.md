# Reading and writing apparatus for ID cards.

## Abstract
A read and or write apparatus for ID cards, includes a magnetic read head 30 for reading at least a magnetic stripe and a magnetic write head 26 are disposed on a running passage for ID cards to be used for an automatic teller system, a credit card system or other various maintenance systems. A connecting member 34 to be connected to a set of connecting terminals 38 of an integrated circuit 36 built into an ID card is arranged, so that read and or write can be effected on three kinds of ID cards, that is, an ID card having only a magnetic stripe, and ID card having both a magnetic stripe and an integrated circuit built therein and an ID card having only an integrated circuit.
