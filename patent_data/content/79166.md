# Slip processing system.

## Abstract
A slip processing system operates to read necessary data from a separate sheet or sheets, and identifies, from the data read, certain letters or numerals chat characterize the slip to be prepared, the apparatus including a device 23 for selecting a predetermined visual maker or mode according to the results of identification, a memory means 22 storing the image of the data read, an editing device 24 for allocating the stored data and selected mark or mode to respective locations on predetermined slip forms, and a printer 28 for printing the edited data onto a prescribed slip sheet 61.
