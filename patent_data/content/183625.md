# Television signal receiving apparatus.

## Abstract
A television signal receiving apparatus having a program schedule display function is disclosed, which includes a tuner 31 , a television signal processing circuit connected to the tuner, a digital signal processing circuit connected to the tuner for generating a program schedule display signal based on a program schedule data obtained through the tuner, and a switch connected to the digital signal processing circuit for selectively generating a local program schedule display signal based on the Standard Time selected by the switch.
