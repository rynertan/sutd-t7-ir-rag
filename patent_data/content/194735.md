# MULTIPLE DISC TYPE FILTER WITH EXTENSIBLE SUPPORT

## Abstract
A multiple disk type filter including a stack of filter disks disposed in a housing and an extensible support for the stack of filter disks, the extensible support including first and second base elements and a multiplicity of rod members joining the first and second base elements in sliding relationship, a first plurality of the rod members being fixedly attached to the first base element and being arranged for slidable supporting engagement with the second base element and a second plurality of the rod members being fixedly attached to the second base element and being arranged for slidable supporting engagement with the first base element.
