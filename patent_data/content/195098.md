# System for reproducing information in material objects at a point of sale location.

## Abstract
The present invention contemplates a system for repro ducing information in material objects at a point of sale location wherein the information to be reproduced is pro vided at the point of sale location from a location remote with respect to the point of sale location, an owner authoriza tion code is provided to the point of sale location in response to receiving a request code from the point of sale location requesting to reproduce predetermined information in a material object, and the predetermined information is repro duced in a material object at the point of sale location in response to receiving the owner authorization code.
