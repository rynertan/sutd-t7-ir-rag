# Apparatus for projecting slides carrying map information on several scales.

## Abstract
Map projecting device comprising a memory for storing relative position and scale factor information, memory reading means, hand operated means for adjusting part of a map projected, switching means for switching between parts of a map or to another map, said maps having of different scales, and a displacement means. When moving from one map to another by operating said switching means any position on said one map is uniquely related to the corresponding position on any one of the other maps. The projection system comprises one single lens unit having a fixed magnification.
