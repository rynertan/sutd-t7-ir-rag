# Information processing unit.

## Abstract
The present invention relates to an information process ing unit capable of performing processing with decreased pro cessing time. The unit comprises first memory means 100 for storing a plurality of instruction words and a plurality of data, second memory means 202 for temporarily storing one of said instruction words read out of said first memory means 100 , numerical data generating means 205 for generating numerical data when said instruction word stored in said sec ond memory means 202 is a first specific instruction word and a first field contained in the first specific instruction word demands generation of data, and logical data generating means 206 for generating logical data when said instruction word stored in said second memory means 202 is said first specific instruction word or a second specific instruction word and a second field contained in said first or second specific in struction word demands generation of data.
