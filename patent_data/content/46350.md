# Method of actuating a plasma display panel.

## Abstract
A method of actuating a PDP device. The method uses an actuating pulse signal which comprises an erasing mode for erasing data displayed on a display panel, a writing mode for writing new data on the panel and a sustaining mode for sus taining a state of displaying the data. The data to be displayed are input by an input signal which is not synchronized with said actuating pulse signal. At and before the time of changing the mode of said actuating pulse signal, at least one pulse is elimi nated from said actuating pulse signal.
