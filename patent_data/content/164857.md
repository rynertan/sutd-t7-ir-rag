# A TRANSMISSION

## Abstract
The transmission comprises first and second input shafts 15, 14 which are drivable respectively from independently operable first and second clutches 3, 7 . Gearing is provided which enables drive to be transmitted from the input shafts to an output via a layshaft 25 . Further gearing 39, 40 is provided on a further layshaft 38 for selection of reverse ratio. In reverse ratio the further gearing is in a drive path which includes the first and second input shafts.
