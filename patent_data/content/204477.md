# Cathode for electron tube and manufacturing method thereof.

## Abstract
A cathode for an electron tube in accordance with the present invention comprises a base 2 containing not only nickel as a major element but also a reducing agent a layer 3 of an electron emissive substance which is applied to the base 2 and contains not only an alkaline earth metal oxide as a principal component but also a scandium oxide and a heater 4 for heating the layer 3 .
