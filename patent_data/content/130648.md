# Rust preventive and process for the preparation thereof.

## Abstract
Rust preventive containing as binder agent a talloil pitch esterified by means of an alcohol and having an acid number between 0 and 15 mg KOH g and a process for pre paring a binder agent by esterifying a talloil pitch having an acid number between 17 and 50 mg KOH g with one or more alcohols.
