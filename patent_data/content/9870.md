# Counter timer with incremental indicators.

## Abstract
In a timer or counter for timing to a selected time or counting to a selected count, there is included digital counting means 60 and means for creating an output signal when the counting means counts to a selected number. In this type of device, there is further provided a plurality of two state indi cator elements 22 28 and means 70 responsive to the counting means for changing the state of one of the elements after the timer or counter has progressed through each of several, pre selected portions of the selected number being counted.
