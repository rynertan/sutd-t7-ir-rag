# Means for storing bass rhythm patterns.

## Abstract
In accordance with the present invention, twelve different partials for a bass rhythm pattern are stored in only four columns of a ROM Read Only Memory 40 . The first three columns 26, 28, 30 comprise a straight binary encoding of six different partials including the bass root, as well as no bass, while the fourth column 32 is used for determining which of two octaves is to be played.
