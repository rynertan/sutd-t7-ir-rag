# DISPLAY SUBSYSTEM

## Abstract
A display subsystem having a graphics capability includes a bit map memory for storing bits, each bit representing a dis played pixel. A read only memory stores words, each word rep resentative of a pixel of a selected pattern which is used to fill out an area of the display thereby clearly identifying adjacent areas of the display to the operator. The selected patterns are displayed in a REPLACE, an OR or an EXCLUSIVE OR mode of operation.
