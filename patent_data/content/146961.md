# Image and graphic pattern processing apparatus.

## Abstract
A graphic pattern processing apparatus using a raster scan type CRT is disclosed. The graphic pattern processing apparatus can update one pixel data, translate a logical address to physical address and transfer data in a display memory 13 , at a high speed. The graphic pattern processing apparatus comprises an operation unit 30 including logical address operation unit 310 , physical address operation unit 320 and color data operation unit 330 , and a control unit 20 including a microprogram memory 100 and a microprogram decoder 200 .
