# Improvement to public telephony method and apparatus employing selector equipped normal and trunk call sets.

## Abstract
The method provides for the acquisition by a public telephone set AT of the user s private telephone number and its transfer through centralized means LC along with information pertaining to the call effected, in particular the debiting of the private telephone num ber with the call. The user is assigned an identification card the validation whereof is performed by a reader LS associated with the public set AT , and a personal digital code which the user dials on a selector device associated with the set AT .
