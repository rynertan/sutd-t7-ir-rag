# A PACKAGE

## Abstract
A package comprising an inner box and an outer box and provided with a hanger device for placing the package as an integral unit onto a horizontal support. The hanger device is formed by an extending end of a plate shaped element arranged for insertion between the inner box and the outer box. The plate shaped element is dimensioned for covering the major part of the broad side of the package. There are fixation means formed by said plate shaped element for removably mounting said boxes to said plate shaped element.
