# Word processing system having a formatting bidirectional printer.

## Abstract
A word processing system is provided having a bidirec tional printer with formatting capability. The system includes a word processor 11 having means for outputting a stream of sequential data readable in only one direction. The data comprises blocks of character data representative of the characters to be printed interspersed with blocks of control data representative of the format in which the characters are to be printed each block of control data has an immediately preceding control block identifier. The bidirectional printer includes means 16, 18 for receiving the stream of sequen tial data, means 18, 15 for scanning the stream of data and means 15 for inserting a control block identifier at the end of each control block which identifier corresponds to the identifier preceding the control block, whereby the stream of sequential data is made readable in both the forward and reverse directions within the printer.
