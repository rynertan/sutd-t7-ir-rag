# DISPLAY INTERFACE APPARATUS

## Abstract
An apparatus for interfacing externally generated dis play data with a display terminal so that the externally generated data can be displayed on the display terminal, the apparatus including means for storing the externally gener ated display data, means responsive to the display control signals of the display terminal for generating addresses for accessing the storage means, wherein the access addresses are in synchronism with the display control signals, and means for logically combining the internally generated display data and the synchronized externally generated display data.
