# An output buffer circuit.

## Abstract
An output buffer circuit for use in a bidirectional input output circuit comprising an inverter circuit formed by a pull up load Q₃₁ and a driver transistor Q₃₂ , and a charging circuit Q₃₅ , CT for charging an external input output line 4 for a predetermined time period T₂ when detecting the falling of the potential V₁ of an internal output line 2 .
