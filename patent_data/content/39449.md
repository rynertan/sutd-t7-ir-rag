# A memory refreshing arrangement.

## Abstract
In the refreshing scheme of the memory contents of the non volatile memory device 3 , a refreshing buffer memory is provided in a manner that a data signal of an address to be refreshed is necessarily kept stored in this refreshing address or stored in a refreshing buffer memory, and not only this data signal is stored in the refreshing buffer memory, but also the address of this data signal and the status of the refreshing processes, that is, up to which step of the refreshing routine the refreshing oper ation proceeds, are stored in this refreshing buffer memory. Then, even when the power supply is shut off during the course of the refreshing action, at the next chance of the resumption of the power supply, the refreshing routine can correctly be started from the step on the way in the prece ding refreshing operation, thereby the refreshing operation can always be completed correctly.
