# Paint compositions and fountain brushes containing them.

## Abstract
A paint suitable for use in a fountain brush comprising up to 5 by weight of a pigment dispersed in an aqueous medium containing a suspending agent and a surfactant dispersing agent the paint exhibits advantageous properties used in a fountain brush comprising a brush, a resevoir containing the paint and flow restricting means for regulating the flow of paint. The preferred suspending agent is hectorite.
