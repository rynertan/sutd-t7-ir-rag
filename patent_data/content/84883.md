# Data transmission network.

## Abstract
The invention discloses a data transmission network having a plurality of data transmission stations 10A 10E connected through a loop like transmission line 11 for mutual data transmission, wherein a plurality of data frames, each having data packets 29 36 allocated to the respective stations, are circulated through the transmission line simultaneously, and each station is arranged to send out the data frame to the transmission line immediately after renewing the data of the data packet allocated to that station in the received data frame, and to process the data of the received data frame to be used for the associated equip ments after sending out the data frame.
