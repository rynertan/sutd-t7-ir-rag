# Patient supporting table top in medical examination and therapy apparatus.

## Abstract
A patient supporting table top for X ray inspection apparatus which includes a platelike core member 11 formed by mixing thermosetting resin with a great number of hollow carbon microspheres reinforced with carbon fibres and press heating these materials. A covering member 12 is bonded to each side of the core member by using an adhesive agent such as epoxy resin. The covering member is formed by mixing resin material with carbon fiber network and press heating these materials.
