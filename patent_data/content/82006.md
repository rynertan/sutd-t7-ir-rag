# Method for producing vicinal dihalogenated products and uses thereof, and process for preparing epoxides using the same.

## Abstract
A method for the manufacture of vicinal dihalogenated products from alkenes and alkynes by enzymatic reaction. The respective alkene or alkyne is reacted with a halogenating enzyme, an oxidizing agent and a halide ion source. The invention also provides a process for preparing epoxides using such products and includes various uses of the aforesaid products.
