# Entertainment machines.

## Abstract
A coin freed player operable entertainment machine 1 is connected via a transmission link, such as a telephone line 2 , to a remote control device 3 . The link may be used to transmit programme information to a control system 10 27 of the machine so that the game to be played with the machine can be changed as desired from the remote control location. Alternatively or additionally, the transmission link may be used to transmit information concerning machine operation from the machine to the remote control location.
