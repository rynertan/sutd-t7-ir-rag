# FLOW CHART SYMBOLS FOR COMPUTER PROGRAMMING

## Abstract
There is described an apparatus for assembling a flow chart for use in computer programming, which comprises a plurality of flat display elements made of plastics material, the shape of each element being substantially that of a flow chart symbol or each element having a flow chart symbol printed thereon and the plastics material having the property of adhering to smooth surfaces, a board having a smooth surface, and means, for example a water based ink pen, for writing eraseably on the board and or on the display elements. There is also described a method of assembling a flow chart for use in computer programming, which compris es using the apparatus.
