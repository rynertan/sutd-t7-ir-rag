# ESPALIER SYSTEM AND VINE MANAGEMENT PROGRAMME SUITABLE FOR KIWI FRUIT

## Abstract
A growing system for vines which is particularly applic able to kiwi fruit which uses an espalier arrangement. The system is such that a canopy of branches are supported while at the same time divergent branches which will form part of the canopy in the ensuing season are supported upwardly in a divergent manner above the canopy. The invention also consists in the frames and framing systems.
