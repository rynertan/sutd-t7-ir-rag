# Document reproduction system with two original document feed devices.

## Abstract
A copier includes an automatic document feeder 16 a semi automatic document feeder 14 and a control system coupled to the feeders by busses A to D. A control button 122, when depressed, causes the control system to switch into a suspending commencing mode in which a current copying task employing one of the feeders is sus pended and a further copying task employing the other feeder is entered. At the end of the further task, the current task continues from the point at which it was suspended.
