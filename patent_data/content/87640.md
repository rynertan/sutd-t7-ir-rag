# Control device for a converter.

## Abstract
A control device for a converter used in a DC transmis sion system comprises an automatic current regulator 56A, 56B , an automatic margin angle regulator 42A, 42B and an automatic voltage regulator 30A, 30B . The output of the automatic voltage regulator 30A, 30B is supplied to the automatic margin angle regulator 42A, 42B to vary a margin angle reference of the automatic margin angle reg ulator 42A, 42B .
