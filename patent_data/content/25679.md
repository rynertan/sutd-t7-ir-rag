# Multicolor coating system, process for producing it, process using it to achieve a multicolor pattern on a surface, and articles having such multicolor patterns.

## Abstract
A multicolor coating system involves an aqueous gel composition which is adapted for application on the surface of articles to achieve a sharply defined multicolor pattern effect. The aqueous gel composition consists of an aqueous gel matrix which has dispersed therein one or more immiscible colorant gel phases. The matrix may itself be colored.
