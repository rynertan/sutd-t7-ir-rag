# Method of generating a list of expressions semantically related to an input linguistic expression.

## Abstract
Method of generating a list of expressions synonymic or antonymic to a linguistic expression input on a keyboard 1 consisting in storing in a first memory 15 a vocabulary of linguistic expressions including an index, storing in a second memory 12 a two dimensional array containing bits linking the address codes of the linguistic expressions, comparing the input linguistic expression to the index in the first memory and storing the address code associated with the stored linguistic expression when an equal occurs, accessing the second memory 12 at the address specified by the address code, utilizing the address codes corresponding to the bits stored at the accessed address to access linguistic expressions stored in the first memory 15 , and displaying on a display 19 the input linguistic expression together with the accessed expressions synonymic or antonymic to said input expression.
