# ADJUSTABLE LENGTH MOLD FOR FORMING CONTAINERS

## Abstract
A package forming apparatus produces containers by means of molding thermoplastic material. A mold having a fixed cross section but a variable length mold cavity 52 is provided by a mold core 54 having a bottom ring means outer shell unit 30, 31 in operative association with the mold core 54 but which is movable axially of the mold core to vary the operative length of the mold and a removable top end plate 62 is provided for the mold cavity.
