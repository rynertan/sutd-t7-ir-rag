# An electric display apparatus.

## Abstract
A display panel 21 has a large number of display units 20 arranged in a matrix. A display circuit 22 for the panel 21 has a drive thyristor for each display unit 20 . The thyristors are controlled by a display register 23 which re ceives display data from RAM s 27 via a data trans mitter 28 . Each RAM is mounted on a respective base plate 51 which is detachable from the remainder of the cir cuitry by means of a multi connector. Back up power is avail able trom a battery B mounted on the respective base plate 51 for the RAM, the base plate 51 being protected within a package 52 with a lever 53 for disconnecting the base plate 51 .
