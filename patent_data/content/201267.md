# ROW PROCESSOR FOR BIT MAP DISPLAY

## Abstract
A row processor unit for a bit map, raster scan display per forms per scan line frame buffer management functions. A portion of the frame buffer memory which is not used for dis play contains load decode parameters. These parameters are accessed by the row processor unit during the horizontal blank ing interval and used for display and data movement operations during the refreshing of the following scan line. A pattern memory provides pixel masks for scrolling as well as providing cursor patterns.
