# DOCUMENT PROCESSING APPARATUS

## Abstract
A document processing apparatus has a plurality of resources including a plurality of data output device, a document processor for processing the data through correc tions and editing, a memory for storing the data inputted from the data input means, as well as the corrected and edited data, and data output device for outputting the processed data. The apparatus further has a first detector for detecting, self diagnostically, one or more resources which has an error contained therein, a second detector for detecting at least one substitute resource that may be replaced with the error containing resource as detected by the first detecting means. The apparatus stops its operation when no substitute resource is detected, but continues its operation when a substitute resource is detected. The substitute resource is replaced with the error containing resource.
