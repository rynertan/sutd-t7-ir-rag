# Impact printer with dual helix character print elements.

## Abstract
An impact printer is provided with a print member which comprises a first helical element 11 having a first font of type characters arranged around its periphery this first helical element is rotatably mounted so as to be linearily movable along an axis adjacent to a print medium. The print member further includes a second helical element 12 having a second font of type characters arranged around its periphery, and this second helical element is rotatably mounted along the same axis in a position interspaced with the first helical element so as to be rotatable and linearily movable along the axis independent of the first helical element. The print member further includes means 20, 21, 17, 22 for rotating the first and second helical elements.
