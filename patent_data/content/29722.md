# A processing method for silver halide colour photographic material.

## Abstract
A method of processing an exposed colour photo graphic material is provided in which the photographic mat erial comprises, on a support, at least one hydrophilic colloi dal layer containing a silver halide emulsion and a coupler and a colour developing agent or a precursor thereof in the or another hydrophilic layer, the method comprising expos ing the material in a colour developer containing a colour developing agent which colour developer is replenished with a replenisher containing 0 0.016 mol litre of a colour developing agent.
