# Radiocommunication system in which a receiving station monitors a number of transmitters by time division.

## Abstract
Radiocommunication system in which a receiving sta tion monitors a number of transmitters by time division. System for transmitting, via radiocommunication chan nels, call messages as specified in CCIR Recommendation 493 1. The transmitter transmits every call message, having a duration t, n times with a message frequency r. The receiving station, designed for monitoring, by time division, n calling stations, has a monitoring period Only one receiver is used. The identification signal of the called station will always be recognized during one of the monitoring periods T, after which the monitoring period can be prolonged, if necessary.
