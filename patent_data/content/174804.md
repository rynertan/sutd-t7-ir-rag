# Home control system and interphone system.

## Abstract
A home control system comprises a pair or a plurality pairs of home information transmission path 21, 22 referred to as home bus information outlets 23 provided at said home bus which serve as connection nodes for appliance 12, 16, 17, 18, 19, 20 and a plurality of room monitor controllers 1 hereinafter referred to as RMC each having a built in interface unit which has a control function means peculiar to a particular room and carries out communication control beteen the information outlets and the appliances.
