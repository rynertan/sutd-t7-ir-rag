# Display type dictionary apparatus.

## Abstract
A display type dictionary apparatus includes a letter signal generator 1 . The letter signal generator generates letter signals which correspond to the input letters. A letter page converter 2 is provided for converting the letter signals into a predetermined page. The contents of the predetermined page are displayed on a screen 4 through a page recorder 3 for storing pages. The display type diction ary apparatus permits to display instantaneously the content of each page in response to a word or indexing word.
