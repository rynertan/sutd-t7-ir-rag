# Block for supporting the poles of equestrian fences.

## Abstract
A block 1 for use in supporting one end of an equestrian fence pole. Each face 2a, 2a 2b, 2b 2c, 2c of the block has a groove 3a, 3a 3b, 3b 3c, 3c for receiving one end of a pole. The three dimensions of the block 1 are all different so that a pole can be supported at three different heights.
