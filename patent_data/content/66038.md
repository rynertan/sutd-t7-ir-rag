# METHOD FOR DISPLAYING AND EDITING SPATIALLY RELATED DATA IN AN INTERACTIVE TEXT PROCESSING SYSTEM

## Abstract
Method for displaying and editing spatially related data in an interactive text processing system having a display 14 , a keyboard 10 , a printer 15 and a microprocessor 11 for storing conventional text data in a first format and spatially related data in a second format, consisting in converting a selected portion of the spatially related data in the first format to displayed data in the second format, and interactively editing the converted spatially related data that is displayed as if it were text data in first format.
