# HEATING COOKING DEVICE.

## Abstract
A safety device for a microwave cooking device having an electronic control unit using a microcomputer 47 . A main power source circuit of the cooking device is controlled by the output signal from the microcomputer 47 controlled by input signals from a keyboard 29 . In the main power source circuit, a heat start switch 37 mechanically operated by a cooking start button 32 is provided in addition to the electronic control unit using the microcomputer 47 . The main power source circuit is broken unless the cook start button 32 is operated. Even if the microcomputer 47 is erroneously operated, there is no fear of abnormally oscillat ing the magnetron 42 .
