# Lock for releasably securing two relatively movable closure mobile elements together.

## Abstract
For fixing for example a door, to a frame on which it is mounted, the device includes an engagement member such as a bolt, able to move from an engagement position to a disengagement position and vice versa, control means, for example a key 2 for the engagement member, and one or more elements for transmitting the operation of the control means to the engagement member. Each trans mission element comprises two or more sectors 12, 13 able to move towards and away from the respective trans mission element. Each sector 12, 13 cooperates with a sliding rocker element 8 able to interact with the en gagement member and be controlled sequentially by the control means 2 by way of one or more transmission elements 15 , to allow movement of the engagement member.
