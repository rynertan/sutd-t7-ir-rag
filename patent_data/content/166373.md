# Disk subsystem read write control.

## Abstract
A track on a disk surface of a disk drive 30 is formatted in sectors, each sector having an address portion and a da ta portion. The disk drive 30 generates a byte clock signal which increments a counter. The counter output signals ad dress a read only memory 4 which generates signals to control the address comparison in the address portion and the reading or writing of data bytes in the data portion of the sector.
