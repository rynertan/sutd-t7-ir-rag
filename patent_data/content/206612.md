# TOOL HOLDER ADAPTOR

## Abstract
There is described a tool member 5, 105 and adaptor 1, 101 for automated tool changing in spindle machines, wherein the adaptor 1, 101 has an external sleeve 19, 119 slidable on a spindle member 3, 103 to selectively engage and release a plurality of gripping elements 33, 133 which retain the tool member 5, 105 in the adaptor. The tool member includes a conical surface 35, 135 contacted by the gripping elements to urge the tool member into the adaptor when the gripping elements are engaged. The sleeve 19, 119 and tool member 5, 105 have grooves 41, 141, 47, 147 for automated handling.
