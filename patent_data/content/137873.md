# Variable pitch axial flow fan.

## Abstract
An axial flow fan with adjustable blades 18 has a spind le 16 for each blade mounted in a hub rim 12. Each spindle has associated therewith a replaceable cartridge 58 having a thrust bearing 70 therein and spaced radial bearings 64, 70. In order to service the bearing, the cartridge can simply be disconnected, e.g. by unscrewing from the rim having removed the blade 18.
