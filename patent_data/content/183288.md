# VERSATILELY APPLICABLE COMPUTER

## Abstract
A computer which is portable and transportable by a person and which can be used on a table, has a universally movable computer input device, such as a spherically shaped input member surrounded by sensors, being attached to the computer housing enabling use of the computer input device combination in further operating conditions such as in a vehicle. One or more sensors suitable for mounting on a vehicle and providing operating data of the vehicle can be connected with computer inputs for computing vehicle results during operation of the vehicle.
