# TOWEL CABINET

## Abstract
A towel cabinet incorporates a mechanism adapted to prevent the tension in the towel being withdrawn from exceeding a predetermined value. In one embodiment the cabinet incorporates a stop mechanism 18 25 which oper ates when a predetermined length of towel has been withdrawn, and a friction clutch 42 45 which allows the stop mechanism to be overridden if a user continues to pull strongly on the towel.
