# Tape Cartridge.

## Abstract
The present invention provides a magnetic tape cartridge which includes a cartridge reel which cannot normally be turned until the cartridge is loaded into a tape transport device. It also provides a cartridge system in which the cartridge reel is self centering when loaded into the tape transport device. The cartridge includes a door which can be automatically opened to provide access to tape in the cartridge, but otherwise provides a completely sealed tape container.
