# Image processing apparatus with automatic window processing function.

## Abstract
An image processing apparatus includes frame memor ies 13, 14 for storing information of an image to be displayed a block memory 22 for dividing the information stored in the frame memories into a given number of information blocks and storing the divided information blocks a microcomputer circuit 21A for automatically determining a desired value of each LEVEL and WIDTH according to given contents of the divided information blocks and an image emphasis circuit 15 for window processing information of the image to be displayed accord ing to the determined value of each of the LEVEL and WIDTH.
