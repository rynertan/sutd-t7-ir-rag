# An optical system focus state detector.

## Abstract
An optical system focus state detector for photometric ally analysing light projected from a source 13,37 to an object 4,32 that is to be viewed or recorded via an optical system 1,31 using an off set projection of said light, analysis means 11,12,45,63 to detect light reflected back from said object and analyse its intensity distribution relative to the axis of the optical system.
