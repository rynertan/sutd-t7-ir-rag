# Beverage dispensing machine.

## Abstract
A beverage dispending machine which, in addition to its normal beverage dispensing modes has machine function modes during which time various machine functions and characteristics may be viewed and changed. The normal user beverage selec tion keys 8 and visual display means 2, 10, 12 take on machine functions and characteristics during the machine function modes. Typically the machine function modes enable faults to be display ed, prices of beverages to be changed or beverage characteris tics to be altered.
