# METHOD AND STRUCTURE FOR DISABLING AND REPLACING DEFECTIVE MEMORY IN A PROM DEVICE

## Abstract
Means are provided for replacing a defective row or column of memory in a fuse array PROM which comprises disabling the defective row and programming a redundant row to respond to the address of the defective row.
