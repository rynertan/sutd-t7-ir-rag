# Method for editing document.

## Abstract
A method of editing a document comprises dis playing 70 a text sentence on the screen of a display unit, bounding 72, 74 an area partly removed of the displayed text sentence and reserved for the display of a figure, calculating 47b a configuration which the bounded area has on the document to be printed, dls playing 78 an area having a size in proportion to the configuration on the display screen, drawing 477, 478 the figure within the displayed area, and synthesizing 84, 86 the text sentence and the figure. When drawing the figure on the screen of the dlspaly unit, the user can monitor a print image.
