# A method for operating a virtual storage management system.

## Abstract
Disclosed is a virtual storage managing system in which the storage address is managed by means of virtual address, and the virtual storage area 100 which can be appointed by the virtual address is divided into an address non translation area 110, 120 which does not require the address translation and an address translation area 130, 140, 150, 160, 170 which requires an address translation. The address translation area is divided into an address fixed area FX and an address variable area FL . In the address fixed area FX , the correspondence between the virtual address and the real address is fixed in a 1 1 fashion, whereas, in the address variable area FL , the correspondence is determined at the time of starting of execution of a program and is dismissed when the execution of the program is completed.
