# PORTABLE VIDEO AND PORTABLE COMMUNICATIONS TERMINAL

## Abstract
Portable communication terminal composed of a head set to be worn on the head of the user and a portable transceiver set to be carried by the user. The terminal includes a non electrical optical fiber conductor mounted for transmitting optical images between the headset and the portable transceiver set optical elements disposed at the headset for enabling an image transmitted to the headset from the transceiver set to be viewed by a user wearing the headset and for causing an image of the scene in front of a user to be formed at the transceiver set a television transmitter and receiver and optical elements at the trans ceiver set for supplying to the transmitter images formed at the transceiver set, and for supplying to the transceiver set an image of the video display for transmission via the optical fiber conductor to the headset.
