# Cable sealing apparatus.

## Abstract
A cable end seal and a closure in which the cable end seal provides a barrier for a junction against water, the junction be ing in a cable. First and second end seals 12, 14 are disposed adjacent opposite sides of the junction 4 and placed in com pression thereat, the end seals having an outer convoluted sur face which mate with a variable length convoluted tubing 32 bridging member so as to form an environmental closure for the junction.
