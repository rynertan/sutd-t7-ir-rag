# Self contained portable data entry terminal.

## Abstract
A self contained, portable data entry terminal positioned within a cordless, wand type enclosure. Positioned within the enclosure is a bar code optical reader 12 , signal conditioning electronics 14, 15 , a microprocessor 16 , a memory 19 , and a rechargable battery 22 . The bar code optical reader is a sensor emitter 12 that functions as both a sensor and an emitter to permit the readout of data stored in the memory 19 .
