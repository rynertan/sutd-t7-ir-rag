# Self sealing sterilization bag.

## Abstract
A sterilizable pouch comprising a plastic web and a paper web marginally heat sealed around all sides thereof, a slit in one of said webs for receiving the package contents and an adhesive strip adhered to the web said adhesive strip being positioned to cover and seal both lips of the slit, and along with the web to which it is attached forming an assembly which acts as means for opening the slit for insertion of an article.
