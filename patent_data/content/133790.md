# Disc playback apparatus.

## Abstract
An apparatus is disclosed for reproducing a disc, such as an optical disc, having main digital data, such as video data, stored thereon in successive blocks associated with sub digital data for searching. Each of said blocks comprises a header signal and a block address signal followed by the main digital data. Means are provided for searching the main digital data using the sub digital data, for detecting the header signal, for detecting whether the block address is the required one, and for deriving the the required main digital data.
