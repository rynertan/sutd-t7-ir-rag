# Non volatile memory system for an electronic musical instrument.

## Abstract
An electronic musical instrument having a plurality of stop tablets 10 comprising a readable, writable, non volatibe memory 16 . The memory includes addressing inputs 20 and data input outputs 18 . The data input outputs 18 are coupled with at least selected ones of the stop tablets 10 . Player actuatable memory controls 22 are also provided for selecting either the memory 16 or the selected ones of the stop tablets 10 for control of the instrument. The memory controls 22 include a control 24 for selectively writing into the memory 16 data corresponding to the actuated or unactuated condition of each of the selected ones of the stop tablets 10 .
