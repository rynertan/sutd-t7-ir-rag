# Slide filter apparatus for filtering contaminants from a fluid working material.

## Abstract
A slide plate filter apparatus particularly for plastic stock under pressure in which rigid filter plate sections 30, 31 carrying corresponding sections of filter media 32, 33 are fed in a continuing sequence across the flow path of the plastic stock is provided with off set retainer plates 34, 35 for directing plastic stock which is forced between the filter plate sections into a path which includes filter media, so that no unfiltered plastic stock can pass through the apparatus.
