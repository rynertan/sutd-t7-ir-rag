# Read write bar for multi mode reproduction machine.

## Abstract
A dual mode reproduction machine or copier incorporating a full width read write bar 50 having at least one linear array 41 of LEDs 42 either for selectively illuminating and scanning a document original 11 in a read mode or for ex posing the copier photoreceptor 22 in a write mode. A Selfoc lens array 45 provides a lens element for focusing each LED, the read write bar and lens array being combined in a unitary structure mounted for selective movement between a read and write position.
