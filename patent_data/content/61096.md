# Data processing system for parallel processing.

## Abstract
There is disclosed a data processing system which has plural operation units 6, 7, 8 1, 8 2, 8 3 and can execute plural instructions in parallel. The system also has plural instruction control means 11 L, 11 S, 11 E, 11 F each of which comprises at least two stages R, W , one R for reading source operands from a local storage 7 , and another W for writing a result operand into the local storage 7 . Each instruction control means is provided with specific bank timings E3, E2, L, E1, F3, F2, K, F1 for accessing the local storage 7 .
